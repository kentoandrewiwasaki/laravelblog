# アプリケーションの概要

簡易的なブログアプリです。

# アプリケーションの機能一覧

* ユーザー新規登録機能
* ログイン/ログアウト機能
* ユーザー情報編集機能
* ブログ投稿、カテゴリ、タグのCRUD機能
* 投稿ゴミ箱の設定、削除投稿の復活機能
* ブログのタイトル名での検索機能
* タグ別、カテゴリ別投稿の表示機能

# アプリケーション内で使用している技術一覧

* 言語 : PHP
* フレームワーク : Laravel
* フロントエンド : HTML5/CSS3/JavaScript/jQuery
* データベース : MySQL
* 画像アップロード : Cloudinary
* サーバー : heroku
